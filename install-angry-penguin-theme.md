# Angry Penguin Theme + Icons Installer for Omarchy

**Last updated:** March 2026  
**Tested with:** User-local Papirus install, Omarchy theme folder structure

This script installs:
- The Omarchy theme named `angry-penguin`
- A custom icon theme `Angry-Penguin` based on Papirus-Dark
- Deeporange colored folders
- Custom mimetypes + Papirus actions icons

## Important – Before Running

1. Extract the Angry Penguin theme archive you downloaded
2. `cd` into the extracted folder  
   (the folder that contains a subfolder named `angry-penguin` – usually the case with GitHub zips)
3. Save the script below as `install.sh`
4. Make it executable:

```bash
chmod +x install.sh
